# 🌐 AI Network Monitoring Dashboard

A real-time WiFi monitoring dashboard built using Streamlit that provides insights into network performance, speed, latency, and connected devices.

---

## 🚀 Features

* 📡 WiFi Name & Signal Strength Detection
* ⚡ Download & Upload Speed Test
* ⏱ Latency Monitoring (Ping)
* 📊 Latency History Graph
* 🚀 Speedometer Gauges (Download / Upload / Latency)
* 🖥 Connected Device Detection (IP + MAC)
* 🤖 AI Latency Prediction
* ⚠ Network Anomaly Detection
* 📄 PDF Report Generation
* 🔄 Auto-refresh Dashboard

---

## 🛠 Tech Stack

* Python
* Streamlit
* Plotly
* Pandas
* Speedtest-cli
* Scapy
* Scikit-learn

## ⚙️ Installation

### 1️⃣ Clone the repository

```bash
git clone https://github.com/chethan1236/ai-network-monitoring-dashboard.git
cd ai-network-monitoring-dashboard
```

---

### 2️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

---

### 3️⃣ Run the app

```bash
streamlit run app.py
```

---

## 📊 Usage

1. Open the dashboard in browser
2. Click **Run Network Test**
3. View:

   * Speed (Download / Upload)
   * Latency
   * Graphs
   * Devices connected
